Elimination ​

For example, we have three equations and three unknowns.

{1x+2y+1z=23x+8y+1z=120x+4y+1z=2

Our purpose is to knock out the x part of the equation 2, 3 and y part of the equation 3.

A=[121381041], x=[xyz] and b=[2122]A=[121381041][12102−2041][12102−2005]

We can use elimination matrix to do the elimination. For the case above, E21=[100−310001] and E32=[1000100−21]
So E32(E21A)=U U is the upper triangular matrix.
Matrix multiplication is not commutative, but can be associative. So E32(E21A)=U can be (E32E21)A=U.

You can quickly gain the answer if you know the matrix-multiplication rule. Click here to see the rule.

Pivot is the first non-zero element in each row. For example, In the first step, pivot is A11, and A22 is the pivot in the second step.

After that you can use back substitution to get the equation's solution.

Permutation Matrix ​

For example, permutation matrix P=[010100001] can exchange the first and second row of matrix A, permutation matrix P=[001010100] can exchange the first and third row of matrix A, permutation matrix P=[100001010] can exchange the second and third row of matrix A...

The Inverse of a permutation matrix is its transpose.

You can use matrix multiplication rule to quickly get the inverse matrix.
